# Python Clicker
A simple autoclicker made in Python with Left/Right Mouse Button functionality, random CPS range, and savable settings.

## Requirements
- Python 3
- Pynput Module

## Installation
1. Install Python 3 ([Microsoft Store](https://apps.microsoft.com/detail/9PNRBTZXMB4Z?hl=en-us&gl=US&ocid=pdpshare) or [Python.org](https://www.python.org/downloads/release/python-31312/)).
2. Download the .zip file from the latest [GitHub Release](https://github.com/BionicBen/Python-Clicker/releases).
3. Run the included Download_Python3_Then_Run_This_First_One_Time.bat file to install the Pynput module.
4. Run the included launch.bat file to start the program.
5. Select "1" to set up your initial settings. Use your selected toggle key to start the clicker after completing the initial setup.

## Features
- Toggle Key: Key used to toggle the clicker on or off.
- Mouse Button Held: Determines whether the corresponding mouse button should be held down or not in order to begin clicking after toggled on.
- Mouse Button Selection: Select whether the clicker applies to your Left Mouse Button, Right Mouse Button, or both.
- CPS Mode: Select whether you want a static amount of clicks per second or a random amount between a range of two numbers.
- Saved Settings: Saves your selected settings to a text file that can be loaded the next time you use the program.
